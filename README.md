# MindcareAI — AI-Assisted Personalized Therapy Platform (Documentation + Demo)

MindcareAI is an AI-assisted mental health support platform designed to provide **24/7 empathetic, context-aware conversations**, especially during **crises** (e.g., disasters, conflict, isolation) when access to professional help may be limited.

> **This repository contains documentation and a demo link. Source code is not included.**

## Demo
- Video (real-time responses from our demo): https://lnkd.in/gYPSTBK8

## Motivation
Our goal is to make therapist-like support more accessible and scalable, while prioritizing privacy, safety, and responsible AI practices.

## Key Features (Demo Scope)
- Emotion-aware responses via **text + voice** analysis
- Contextual personalization using **RAG (Retrieval-Augmented Generation)** + vector search
- Iterative improvement with **RLHF (learning from human feedback)**
- Multilingual support, mood tracking, and meditation modules (planned / prototype)

## High-Level Architecture
- Web/Mobile UI for user interaction
- Session and user metadata storage
- Knowledge base (therapy techniques, guidelines, evidence-based practices, etc.)
- RAG pipeline with vector retrieval
- Model orchestration via an agent layer
